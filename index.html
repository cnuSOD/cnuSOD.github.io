CNUSOD Dataset

发布时间：2024-11-1    点击量：1486

     The primary task of building the CNUSOD dataset is to expand the size of the SOD dataset. We collected a large number of high-quality images through Internet collection, manual photography, etc., and finally obtained 120,000+ unlabeled images after manual screening. In the image selection process without annotated data, we appropriately add large-sized salient object images and complex images such as multiple objects to more accurately evaluate the detection performance of the model.。
